👕 Sell My Style

Sell My Style is a work-in-progress C2C fashion marketplace web application where users can buy, sell, and discover clothing items online.

This project is currently under development.

🚧 Status

In Development

🛠️ Tech Stack
React.js (Frontend)
Tailwind CSS (Styling)
Node.js (Backend)
Express.js (Server)
MongoDB (Database)
🎯 Features (Planned)
Browse clothing products
Add and sell clothing items
Product listing with images and details
User authentication (future update)
Responsive UI for all devices
⚙️ Setup
Frontend
cd frontend
npm install
npm run dev
Backend
cd backend
npm install
npm start
📌 Note

This project is still in development and features are being actively added.
